# Handover — per sessie één bestand

**Elke sessie schrijft een eigen, nieuw bestand in deze map** — nooit in een
bestaand bestand. Zo raken twee parallelle sessies nooit dezelfde regel en
ontstaat er **nooit** een merge-conflict.

## Naamgeving
```
docs/handover/<YYYY-MM-DD-HHMM>-<korte-titel>.md
```
Tijd in **UTC**. Hoogste bestandsnaam = nieuwste sessie.

## Lezen
Sorteer aflopend op bestandsnaam, nieuwste eerst. Deze map ís de log.
